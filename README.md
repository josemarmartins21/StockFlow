# StockFlow
Sistema de gestão de estoque

## Como intalar o sistema?
### Roda o comando a seguir:

```
composer install
```

Renomeia o arquivo **.env.example** para **.env**
### Roda as migrations com:
```
php artisan migrate
```
### Excute o projecto com estes comandos abaixo:

```
php artisan serve
```

```
npm run dev
```
